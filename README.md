#  Análise de RNA SARS-CoV-2

Projeto de bioinformática focado na análise genômica das variantes do vírus SARS-CoV-2.

##  Estrutura do Projeto

- `analise_evolutiva.py` → análise das mutações entre variantes
- `analise_visual.py` → visualização dos dados genéticos
- `recortar.py` → processamento e corte das sequências
- Arquivos `.fasta` → sequências genéticas das variantes:
  - Wuhan
  - Alfa
  - Delta
  - Ômicron

##  Tecnologias utilizadas

- Python
- Bioinformática (análise de sequências)

##  Como executar

1. Instale o Python
2. Execute os scripts:

```bash
python analise_evolutiva.py
python analise_visual.py
